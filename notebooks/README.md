# Notebooks Directory

This directory contains Jupyter notebooks for testing, experimentation, and analysis.

## Notebooks

### TestRun.ipynb
- Testing and experimental runs
- Quick prototyping
- Data analysis

## Usage

### Start Jupyter:
```bash
jupyter notebook
```

### Or use JupyterLab:
```bash
jupyter lab
```

## Requirements

Install Jupyter if not already installed:
```bash
pip install jupyter notebook jupyterlab
```

## Notes

- Notebooks are for development and testing only
- Don't commit sensitive data in notebooks
- Clear output before committing
- Keep notebooks organized and documented
