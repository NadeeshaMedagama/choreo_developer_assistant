Problem: "ngrok: command not found"
Solution: Run STEP 1 again, then restart your terminal or run:
          source ~/.bashrc

Problem: "ERR_NGROK_108: Invalid authtoken"
Solution: Run STEP 2 again with the correct authtoken

Problem: "tunnel session failed: failed to connect to ngrok"
Solution: Check your internet connection

Problem: "bind: address already in use"
Solution: Another ngrok instance is running. Close it first.

Problem: Can't access the ngrok URL
Solution: Make sure your application is running BEFORE starting ngrok
